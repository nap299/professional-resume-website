# 🚀 IT Resume — Wiroad Bunsree (NAP)

A modern, interactive portfolio & resume website built with cutting-edge web technologies. Designed to showcase professional experience, technical skills, and project portfolio in a visually stunning presentation.

## ✨ Features

- **Immersive 3D Background** — WebGL particle system powered by Three.js, reactive to mouse movement
- **Smooth Animations** — GSAP ScrollTrigger + Lenis smooth scrolling for a premium feel
- **Interactive Portfolio** — Dynamic project gallery with modal previews and category tags
- **Print-Ready Resume** — One-click A4 PDF export with clean, professional formatting
- **Responsive Design** — Fully optimized for desktop, tablet, and mobile devices
- **Social Integration** — Direct links to Telegram, Gmail, and WeChat QR code modal

## 🛠️ Tech Stack

| Category | Technologies |
|----------|-------------|
| **Frontend** | HTML5, CSS3, JavaScript (ES6+) |
| **UI Framework** | Tailwind CSS, Alpine.js |
| **3D Graphics** | Three.js (WebGL) |
| **Animation** | GSAP, ScrollTrigger, Lenis |
| **Icons** | Lucide Icons |
| **Fonts** | Space Grotesk, Fira Code (Google Fonts) |

## 🚀 Quick Start

1. **Clone** the repository
2. **Open** `index.html` in your browser, or serve with any HTTP server:
   ```bash
   # Python
   python -m http.server 8888

   # Node.js
   npx serve .

   # PowerShell
   Start-Process "index.html"
   ```
3. **Print Resume** — Click the 🖨️ Resume button in the navbar to export as PDF

## 📄 Resume Export

Click the **Resume** button in the navigation bar to generate a clean, A4-formatted PDF resume containing:
- Profile photo & contact information
- Professional summary
- Work experience & education
- Technical skills
- Selected project highlights
